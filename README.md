# 红白字生成器 客户端版 + 网页版

## 简介

这个项目的核心模块来源于https://github.com/rare25/5000choyen这个项目原作者的JS模块，通过F12查看源代码拿到功能，并使用Node.js + Electron重构。

项目将同时发布客户端版本以及网页版。
